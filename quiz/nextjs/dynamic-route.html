動的ルート

動的ルートとは?

解答

動的ルートはページ生成時に動的に設定されるルートのことで, pagesディレクトリ以下でファイル名を[]で囲ったコンポーネントに適用されます

例えばpages/quiz/[pid].jsとすると大括弧の部分は動的に決定されます. pidが1ならquiz/1となります.

また, pidに渡された値はNext.jsのルーターが持つqueryオブジェクトを通じて参照できます. 

                    
                        import { useRouter } from 'next/router'

                        const Quiz = () => {
                        const router = useRouter()
                        const { pid } = router.query

                        return 
Quiz: {pid}

                        }

                        export default Quiz
