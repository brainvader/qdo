React要素

React要素とは?

解答

React要素はcreateElement関数の戻り値です. 単なるPOJO(Plaing Old Javascript Object)で, 実際に描画されるDOMの設計図とも言えます.

以下のような引数でcreateElementを実行した場合を考えます.

                    
                        React.createElement(
                            'h1',
                            {className: 'qdo'},
                            'Let's Qdo!'
                            )

するとReact要素が生成されます.

                    
                        {
                            type: 'h1',
                            props: {
                              className: 'qdo',
                              children: 'Let's Qdo!'
                            }
                        }
