コンポーネント

コンポーネントとは?

解答

コンポーネントはReactアプリケーションを構成する中心的な構成要素です. コンポーネントには実装方法によって関数コンポーネントとクラス・コンポーネントがあります.

関数コンポーネントの場合は以下のように定義します. 

                    
                        function Hello(props) {
                            return (

                            )
                        }

見た目はHTMLタグとよく似ています. 属性はpropsと呼ばれます. propsはpropertiesの略でオブジェクトとしてコンポーネントに渡され内部で利用されます. propsはただ外部から与えられるだけで絶対に内部で変更してはいけません. 全てのコンポーネントはpropsに対して純粋関数として振舞うべきとされています.

もう一つステートと呼ばれるコンポーネントの内部状態を表すプロパティを持ちます. ステートを持つかどうかでステートフル・コンポーネントとステートレス・コンポーネントに分けられます.
