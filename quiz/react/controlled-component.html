制御コンポーネント

制御コンポーネントとは?

解答

制御コンポーネントはHTMLのinput要素, textarea要素, select要素などのそれ自身が管理している状態をReactのコンポーネントで管理する手法です.

例えばinput要素はvalueという値にアクセスすることで入力された値を取得できます. changeイベントにイベント・リスナを登録しておくと入力の変化するたびに値を取得できます. 

                    
                        const changeListener = (event) => console.log(event.target.value); 
                        // input要素を取得しておく
                        inputElement.addEventListener('change', changeListener)

この値をReactの状態として管理します. 

                    
                        function ControlledComponent() {
                            const [value, setValue] = useState('')

                            const changeListener = event => setValue(event.target.value);

                            return (
                                
                            );
                        }
