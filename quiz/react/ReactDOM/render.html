ReactDOM.render

ReactDOM.renderとは?

解答

ReacDOM.render関数は受け取ったReact要素をルートDOMノード以下に描画する関数で, コンポーネントへの参照を返します*1. 

                    
                        const element = createElement( /* your element info */ )
                        const rootNode = document.getElementById('root')
                        ReactDOM.render(element, rootNode)

オプションでコールバック関数を与えることができる. この第3引数はコンポーネントが描画された後, あるいは更新された後に実行されます.

脚注

  1. コンポーネントが非同期で描画されるようになる場合があるらしく, この挙動はレガシーになるようです.
