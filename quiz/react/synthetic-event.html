合成イベント

合成イベントとは?

解答

合成イベントとはクロスブラウザのネイティブ・イベントに対するラッパーです.

ブラウザによってブラウザ・イベントの挙動が違う. 合成イベントによってその違いを吸収している.

利用するのは簡単で単にイベント・ハンドラ に適当な名前で渡せばよい. 

                    
                        function handleClick(e) {
                            e.preventDefault();
                            // 何かする
                        }

handleClick関数の引数eが合成イベントということになる.
