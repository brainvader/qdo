残余引数構文

残余引数構文とは?

解答

残余引数構文は可変個引数を指定できる構文です. 残余と呼ばれるのは第2引数以下に指定することで残り全部の引数を表現できるからです.

引数の前に3点リーダ*1を付けると残余引数として扱われ, 配列になります. 残余引数の後ろに引数を置くことはできません. 

                    
                        function sum(a, b, ...rest) {
                            const initialValue = a + b;
                            return rest.reduce(pre, current) => {
                            }, initialValue);
                        }

sum関数に渡された第3引数以降は例え1つしか値を指定しなくてもrestに配列として格納されます.

脚注

  1. 同様に3点リーダを使う構文に スプレッド構文があります.
